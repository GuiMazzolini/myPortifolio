What I do

Frontend

Frontend development is the part of web development that focuses on creating the user interface and experience of a website or web application. I use tools such as HTML, CSS, JavaScript, React, and Bootstrap to build and design the visual components that users interact with.

Backend

Backend development is the part of web development that focuses on the server-side of a website or web application. I use tools such as Node.JS, Express.JS, Handlebar, and Axios to build the logic and functionality of the application, including database management, API development, and server configuration.

Full Stack

Fullstack development is the combination of frontend and backend development, where developers handle the entire process of building a web application from start to finish. In this area, I am responsible for designing, developing, testing, and deploying the application.
